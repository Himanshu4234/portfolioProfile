## Key Features

- **SEO-Driven Architecture**  
  Integrated metadata, OpenGraph tags, and semantic HTML to boost search visibility and organic reach.

- **Modular Component System**  
  Built reusable components for:
  - Services  
  - Solutions  
  - Industries  
  - Contact sections  

  Enabling the team to add content without code rewrites.

- **SSR-inspired Rendering & ISR Concepts**  
  Optimized TTFB and improved indexing by structuring pages using prerender strategies suitable for static hosting.

- **High-Performance UI**  
  Lazy-loading, optimized image loading, and code splitting improved load time significantly.

- **Responsive & Mobile-First Design**  
  Ensured consistent UX across desktop, tablet, and mobile devices.

- **Clean and Lightweight Architecture**  
  Designed a component-driven folder structure for long-term scalability.
